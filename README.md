## Overview
LawyerGPT is an AI-powered legal assistant that answers questions by **retrieving and referencing the Indian Constitution** using Retrieval-Augmented Generation (RAG).

The system indexes the full Constitution PDF, performs semantic search on relevant articles, and generates accurate, context-aware legal answers.

This ensures responses are **grounded in real constitutional text**, not hallucinated.

---

## Data Source
Primary Knowledge Base:
- Indian Constitution (PDF)

All answers are generated by retrieving relevant constitutional articles and using them as context for the language model.

---

## How It Works

1. Load Constitution PDF
2. Split into semantic chunks
3. Generate embeddings
4. Store in FAISS vector database
5. On query:
   - Retrieve most relevant Articles
   - Generate answer using retrieved context
